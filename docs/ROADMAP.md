# Roadmap

Termode is stabilizing as a standalone Android terminal before larger runtime
or IDE integration work.

## Near Term

- v0.38 Documentation / Onboarding Polish
- v0.39 UI / Settings Polish (current)
- v0.40 Beta Candidate

## Product First

Current focus:

- clearer onboarding
- compact help and examples
- reliable REAL PTY behavior
- package/workspace/session QA
- terminal UX polish
- honest doctors and known-limit messaging

## Later

- CalypsoIDE integration
- Node.js/npm research
- Python/Git research
- native binary package strategy
- richer package ecosystem

Runtime work remains deferred until the standalone Termode experience is stable.
